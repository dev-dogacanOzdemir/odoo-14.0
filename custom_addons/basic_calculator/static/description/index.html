Overview

Everyone needs a calculator.

Basic Calculator is a very useful tool that can help you solve equations and arithmetic problems that you face in your daily work. It can perform quick math operations like addition, subtraction, multiplication and division while using your apps.

Features

Do basic arithmetic operations such as addition, subtraction, multiplication and division

Popup calculator that you can use anytime while using other apps

Use the number pad to do your operations

Re-use old answers in your next calculation
